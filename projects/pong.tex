\resumeItem{\href{https://github.com/sachiniyer/pong-wasm}{pong-wasm}}
{A RL model for pong trained with \href{https://spinningup.openai.com/en/latest/spinningup/rl_intro3.html}{policy gradients} built with \href{https://github.com/huggingface/candle}{candle}, compiled to \href{https://webassembly.org/}{wasm}, and runs in your browser. Utilized web features like \href{https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API}{IndexedDB}, \href{https://developer.mozilla.org/en-US/docs/Web/API/Web_Workers_API/Using_web_workers}{Web Workers} and \href{https://docs.rs/serde-wasm-bindgen/}{custom serialization} with hacks into \href{https://github.com/rustwasm/wasm-bindgen/}{wasm-bindgen}.}
